# Re!tu & Co - Professional Website

A professional website for Re!tu & Co digital artistry services.

## Features
- Responsive design
- Contact forms
- Project submission system
- Portfolio gallery
- Admin dashboard

## Pages
- Homepage
- Services
- Portfolio
- Contact

## Technology Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: PostgreSQL (Neon)
- Deployment: Vercel

## Setup
1. Clone repository
2. Install dependencies: `cd backend && npm install`
3. Set environment variables
4. Deploy to Vercel

## Environment Variables
- `DATABASE_URL`: Neon PostgreSQL connection string
- `NODE_ENV`: production
